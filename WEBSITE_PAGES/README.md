# Website Pages

This directory contains documentation for the static pages deployed to thespacedata website.

## Pages

- **Privacy Policy** (`/privacy-policy/`) - Comprehensive privacy policy covering data collection, usage, and user rights
- **Support** (`/support/`) - Support page with FAQs, contact information, and help resources

## Features

- Responsive design using Tailwind CSS
- Consistent styling with the main site
- Footer links for easy navigation
- SEO-friendly structure

## Deployment

Pages are automatically deployed via GitHub Actions when changes are pushed to the `main` branch.

See `DEPLOYMENT.md` for detailed deployment instructions.
